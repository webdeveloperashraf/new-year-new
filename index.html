Description of Webdeveloper

Web developers build websites: they use code to achieve the right functionality and aesthetic look. Software engineers build programs, software and applications, and are less involved with customer-facing sites.

1. Programmer vs. Developer - These terms are one and the same. Whether you have Programmer or Developer on your business card.
2. Front End Engineer - “Front end” includes all customer-facing features of a website or app: text fields, images, buttons and banners. These engineers program the interface users engage with. HTML, JavaScript are all front end coding languages.
3. Back End Engineer - Back end engineers build the mechanics of how a site processes information. They determine how the site will communicate with servers, and how to collect, store and manage user data.
4. Full Stack Engineer - If you know multiple languages and have successfully built both front and back ends of a site, congrats: you’re a full stack engineer.
5. Software Architect - An experienced programmer who leads the development team. They make high-level decisions about design and functionality.
